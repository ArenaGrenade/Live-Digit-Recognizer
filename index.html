Live Digit Recognizer

Built with by Rohan Asokan
Just draw any digit into the box on the left and the numbers on the right would automatically adjust to reflect the number you drew. It works using a 99.4% accurate Machine Learning Model built using Tensorflow and trained over the MNIST dataset.
There might be a tiny delay/lag the first time the model predicts. This is normal and happens just once. I speedran the project in under 4hrs, if there are any bugs do report them.
Left Click to draw. Right Click to erase.
